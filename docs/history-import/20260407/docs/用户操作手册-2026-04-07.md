# 用户操作手册（简版）

> 文档日期：2026-04-07
> 适用角色：管理员、房东

## 1. 登录与导航

- 访问登录页并输入账号密码。
- 登录后通过顶部导航进入：房产、合同、支付与账单、报表、通知。

## 2. 日常业务流程

### 2.1 房产管理

- 新增房产：填写名称、地址、租金、状态。
- 编辑房产：在列表点击编辑并保存。
- 删除房产：确认无依赖后删除。

### 2.2 合同管理

- 创建合同：选择房产、填写租客信息、租期、租金。
- 查看合同：列表进入详情页确认状态。
- 到期提醒：进入到期提醒页执行提醒或续约。

### 2.3 支付与账单

- 生成账单：在支付列表按账期生成。
- 记账收款：打开账单详情录入收款。
- 查看收据：通过“查看”按钮进入收据页。

### 2.4 报表分析

- 财务报表：支持按账期、支付方式、逾期层级筛选。
- 月度对账：支持趋势、排序、导出。
- 导出 CSV：点击“导出CSV”按钮获取数据。

## 3. 移动端与弱网模式

- 移动端支持表格卡片化与操作区换行。
- 在财务报表/对账页可勾选“弱网优先模式”，折叠高负载区块。

## 4. 常见操作建议

- 筛选无结果时先点重置。
- 使用“常用预设”快速定位待收/逾期数据。
- 使用“条件记忆”减少重复输入。
