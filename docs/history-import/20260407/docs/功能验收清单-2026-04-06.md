# 功能验收清单（2026-04-06）

> 文档状态：历史（归档参考）
> 口径说明：以 docs/主任务看板.md 与 docs/任务清单.md 为准。


## 一、验收范围

- 认证与权限主链路
- 房产管理 CRUD（含筛选/分页）
- 合同管理 CRUD（含状态字段）
- 合同到期提醒雏形
- 收款登记闭环（生成/逾期/记账/收据）
- 基础测试门禁（Unit/Integration）

## 二、验收项

### 1. 认证与权限
- [x] 未登录访问受保护路由会被拦截并跳转登录页
- [x] 登录成功后可访问仪表板
- [x] 登出后再次访问受保护路由会被拦截
- [x] 路由中间件 `auth/guest/api` 已生效

### 2. 房产管理
- [x] 房产列表可读取数据库真实数据
- [x] 支持关键字筛选与状态筛选
- [x] 支持分页展示
- [x] 支持新增/编辑/删除房产
- [x] 房产操作具备所有者/管理员权限控制
- [x] 表单具备 CSRF 校验与基础字段校验

### 3. 合同管理
- [x] 合同列表可读取数据库真实数据
- [x] 支持按关键词/状态筛选
- [x] 支持新增/编辑/删除合同
- [x] 支持合同详情页查看
- [x] 支持合同状态字段（pending/active/expired/terminated）
- [x] 合同表单具备基础业务校验（日期、金额、付款日等）

### 4. 到期提醒（雏形）
- [x] 已提供 `/contracts/expiring` 页面
- [x] 支持按天数查询即将到期合同
- [x] 页面可生成续约提醒建议文案

### 5. 质量门禁
- [x] Unit + Integration 测试通过（36/36）
- [x] 关键高优先级缺陷已修复并记录

### 6. 收款登记闭环（P1-A）
- [x] 支持按周期批量生成账单
- [x] 支持逾期状态刷新与滞纳金计算
- [x] 支持线下收款记账（cash/bank_transfer/alipay/wechat_pay）
- [x] 支持收据页面查看与打印
- [x] 支持房东仅访问自有账单（越权访问返回 404）
- [x] 支持房东列表仅看到自有账单（owner 过滤生效）
- [x] 支持按账单状态与账期筛选列表（status/period 过滤生效）
- [x] 支持支付记账边界状态判定（partial/paid）
- [x] 支持逾期账单含滞纳金的记账边界判定（overdue + late_fee）
- [x] 支持水电公式账单（上月读数/本月读数/当月用量/单价/费用）
- [x] 支持“水电费用 + 固定租金”合并计算月度应收总额
- [x] 支持创建页自动带出上期读数与实时预估总额提示
- [x] 支持批量生成时自动继承上期水电读数与单价
- [x] 支持账单 CSV 导出（用于月度统计与归档）

## 三、暂未纳入本轮验收

- 移动端能力

## 四、验收结论

- Day 1 ~ Day 7 + P1-A 已达到“可演示、可回归、可继续迭代”的状态。
