# 高优先级缺陷清单（2026-04-06）

> 文档状态：历史（归档参考）
> 口径说明：以 docs/主任务看板.md 与 docs/任务清单.md 为准。


## P0-1 启动链路重复执行风险
- 问题：bootstrap.php 在被入口文件 include 时会自动 init+run，可能导致请求重复分发。
- 影响：响应重复输出、副作用重复执行、调试困难。
- 修复：仅在直接执行 bootstrap.php 时自动运行。
- 修复文件：bootstrap.php
- 状态：已修复

## P0-2 默认数据库凭据硬编码
- 问题：核心配置存在默认用户名/密码硬编码（含明文）。
- 影响：安全风险高，且易在配置缺失时误连错误数据库。
- 修复：将默认用户名/密码置空，通过环境配置或数据库配置文件注入。
- 修复文件：app/Core/Application.php, app/Core/Config.php
- 状态：已修复

## P1-1 入口配置覆盖不完整
- 问题：public/app.php 仅加载 app 配置，导致 database 配置可能退回默认值。
- 影响：运行环境切换时数据库连接不稳定。
- 修复：入口同时加载并映射 mysql 连接配置到 application init 配置。
- 修复文件：public/app.php
- 状态：已修复
