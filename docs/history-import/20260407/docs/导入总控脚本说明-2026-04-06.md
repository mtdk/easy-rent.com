# 导入总控脚本说明（2026-04-06）

> 文档状态：历史（归档参考）
> 口径说明：以 docs/主任务看板.md 与 docs/任务清单.md 为准。


## 1. 脚本位置

- scripts/run-import-pipeline.sh

## 2. 作用

一条命令串联执行以下阶段：

1. 生成 xlsx 预览
2. 生成 dry-run 报告
3. commit 模式下自动先备份数据库
4. 按需执行 bootstrap（房源/合同）
5. 导入可匹配账单
6. 解析并处理歧义账单
7. 输出验收快照汇总

## 3. 使用方法

```bash
cd /opt/homebrew/var/www/easy-rent.com
./scripts/run-import-pipeline.sh [mode] [xlsx_path] [db_user] [db_password] [db_name] [db_host] [db_port]
```

参数：
- mode: preview | commit（默认 preview）
- xlsx_path: 默认 docs/A.xlsx

示例（预演，不写库）：

```bash
./scripts/run-import-pipeline.sh preview docs/A.xlsx xmtdk 12345678 easy_rent 127.0.0.1 3306
```

示例（执行写库）：

```bash
./scripts/run-import-pipeline.sh commit docs/A.xlsx xmtdk 12345678 easy_rent 127.0.0.1 3306
```

## 4. 输出产物

- 预览：storage/import-preview/a-xlsx-preview.json
- dry-run：storage/import-preview/a-xlsx-dry-run-report.json
- bootstrap 结果：storage/import-preview/a-xlsx-bootstrap-commit-result.json
- 歧义解析：storage/import-preview/a-xlsx-ambiguous-resolved-report.json
- 账单导入结果：storage/import-preview/a-xlsx-payments-commit-result.json
- 总控汇总：storage/import-preview/a-xlsx-pipeline-summary.json
