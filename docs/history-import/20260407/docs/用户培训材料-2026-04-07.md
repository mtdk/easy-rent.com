# 用户培训材料（管理员/房东）

> 文档日期：2026-04-07
> 培训对象：管理员、房东
> 培训目标：完成核心业务闭环并掌握异常处理路径

## 1. 培训安排

- 总时长：90 分钟
- 结构：
  - 20 分钟：系统概览与权限说明
  - 40 分钟：核心操作实操
  - 20 分钟：常见问题与异常处理
  - 10 分钟：考核与答疑

## 2. 核心实操清单

### 2.1 房产与合同

- 新增房产并确认列表可见。
- 创建合同并检查状态与详情页。
- 进入合同到期提醒页执行提醒动作。

### 2.2 账单与收款

- 生成当期账单。
- 录入收款并查看收据。
- 在支付列表中使用筛选预设与条件记忆。

### 2.3 报表与对账

- 查看财务报表并导出 CSV。
- 在移动端查看摘要卡与卡片化表格。
- 开启弱网优先模式（lite）验证可读性。

## 3. 常见问题

- 问题：页面打不开或跳登录。
  - 处理：检查会话是否过期，重新登录。
- 问题：筛选结果为空。
  - 处理：点击重置，再按单一条件逐步筛选。
- 问题：导出文件乱码。
  - 处理：确认未关闭 BOM（默认开启）。
- 问题：移动端页面加载慢。
  - 处理：开启 lite 模式，折叠高负载区块。

## 4. 培训考核标准

- 能独立完成：
  - 房产新增 + 合同创建
  - 账单生成 + 收款登记 + 收据查看
  - 财务报表筛选 + 导出
- 能说出异常上报最少信息：
  - 页面路径
  - 操作步骤
  - 报错截图
  - 发生时间

## 5. 培训后交付

- 培训签到表
- 实操完成截图
- 5 分钟回访问卷（易用性/稳定性/改进建议）
