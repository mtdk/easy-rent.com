## 验证步骤

### 1. 环境验证
```bash
# 检查PHP版本
php --version

# 检查MariaDB版本
mysql --version

# 检查Composer
composer --version
```

### 2. 数据库初始化
```bash
# 运行数据库初始化脚本
php scripts/init-database.php

# 或使用强制模式（删除现有数据库）
php scripts/init-database.php --force
```

### 3. 应用测试
```bash
# 启动开发服务器
php -S localhost:8000 -t public

# 访问测试页面
# http://localhost:8000/test.php

# 访问应用首页
# http://localhost:8000/
```

### 4. 功能验证
1. 访问应用首页，查看Bootstrap资源是否正常加载
2. 测试登录表单前端验证
3. 检查控制台无JavaScript错误
