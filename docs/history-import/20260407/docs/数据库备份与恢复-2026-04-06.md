# 数据库备份与恢复说明（2026-04-06）

> 文档状态：历史（归档参考）
> 口径说明：以 docs/主任务看板.md 与 docs/任务清单.md 为准。


## 1. 目的

为 easy_rent 提供可重复执行的一键备份与恢复流程，建议在以下时点执行备份：

- 导入前
- 导入后
- 批量修复前
- 重大发布前

## 2. 备份脚本

- 脚本路径：scripts/backup-easy-rent.sh
- 输出目录：storage/backups
- 输出文件命名：easy_rent_backup_YYYYMMDD_HHMMSS.sql

## 3. 备份命令

```bash
cd /opt/homebrew/var/www/easy-rent.com
./scripts/backup-easy-rent.sh xmtdk 12345678 easy_rent 127.0.0.1 3306 storage/backups
```

参数顺序：
1. db_user
2. db_password
3. db_name
4. db_host
5. db_port
6. output_dir

## 4. 恢复命令

示例（恢复指定备份文件）：

```bash
cd /opt/homebrew/var/www/easy-rent.com
mysql -h127.0.0.1 -P3306 -uxmtdk -p12345678 easy_rent < storage/backups/easy_rent_backup_20260406_021044.sql
```

注意：
- 恢复操作会覆盖当前库中的同名对象，请先确认是否需要先备份当前状态。
- 建议先在测试库验证恢复结果，再对生产数据执行。
