## 1. 用户认证与权限管理模块（PHP实现）

### 1.1 功能概述
提供安全的用户认证、授权和会话管理功能，支持管理员和房东两级权限控制，预留租客扩展接口。

### 1.2 核心功能
- **用户注册**：仅管理员可创建房东账号，无租客注册
- **用户登录**：账号密码登录，Session管理
- **权限管理**：两级权限控制（管理员、房东）
- **会话管理**：PHP Session，安全Cookie
- **安全功能**：密码强度检查，登录失败锁定，bcrypt哈希
- **扩展预留**：代码层面预留租客认证接口

### 1.4 PHP实现要点
```php
// 用户认证类示例（支持两级权限，预留租客扩展）
class Auth {
    private $db;
    private $session;
    
    // 角色常量
    const ROLE_ADMIN = 'admin';
    const ROLE_LANDLORD = 'landlord';
    const ROLE_TENANT = 'tenant'; // 预留，当前版本未使用
    
    public function __construct($db) {
        $this->db = $db;
        session_start();
        $this->session = &$_SESSION;
    }
    
    public function login($username, $password) {
        // 查询用户（排除tenant角色，当前版本不支持）
        $stmt = $this->db->prepare(
            "SELECT * FROM users 
             WHERE username = ? AND is_active = 1 AND role IN ('admin', 'landlord')"
        );
        $stmt->execute([$username]);
        $user = $stmt->fetch(PDO::FETCH_ASSOC);
        
        if ($user && password_verify($password, $user['password_hash'])) {
            // 登录成功，设置Session
            $this->session['user_id'] = $user['id'];
            $this->session['user_role'] = $user['role'];
            $this->session['logged_in'] = true;
            
            // 更新最后登录时间
            $this->updateLastLogin($user['id']);
            return true;
        }
        return false;
    }
    
    public function isLoggedIn() {
        return isset($this->session['logged_in']) && $this->session['logged_in'];
    }
    
    public function isAdmin() {
        return $this->isLoggedIn() && $this->session['user_role'] === self::ROLE_ADMIN;
    }
    
    public function isLandlord() {
        return $this->isLoggedIn() && $this->session['user_role'] === self::ROLE_LANDLORD;
    }
    
    // 预留租客检查方法
    public function isTenant() {
        // @todo 未来扩展：租客功能
        return false;
    }
    
    public function canAccessModule($module) {
        if (!$this->isLoggedIn()) return false;
        
        $role = $this->session['user_role'];
        
        // 权限矩阵：管理员可以访问所有，房东有限制
        $adminModules = ['user_management', 'system_settings', 'all_properties', 'all_contracts'];
        $landlordModules = ['own_properties', 'own_contracts', 'payments', 'reports'];
        
        if ($role === self::ROLE_ADMIN) {
            return in_array($module, $adminModules) || in_array($module, $landlordModules);
        } elseif ($role === self::ROLE_LANDLORD) {
            return in_array($module, $landlordModules);
        }
        
        return false;
    }
}
```

## 2. 房产管理模块

### 2.1 功能概述
管理房东的房产信息，包括房产基本信息、图片、设施、状态等，支持本地图片上传。

### 2.2 核心功能
- **房产信息管理**：CRUD操作，批量导入导出（Excel/CSV）
- **图片管理**：多图上传，本地存储，图片压缩
- **地址管理**：文本地址存储，无地图服务
- **状态管理**：空闲、已租、维修中、已售等状态
- **权限控制**：管理员查看所有，房东仅查看自己的房产

### 2.4 PHP权限控制实现
```php
class PropertyController {
    private $auth;
    private $db;
    
    public function __construct($auth, $db) {
        $this->auth = $auth;
        $this->db = $db;
    }
    
    public function index() {
        if (!$this->auth->isLoggedIn()) {
            redirect('/login');
        }
        
        // 根据角色获取房产列表
        if ($this->auth->isAdmin()) {
            // 管理员：获取所有房产
            $stmt = $this->db->query(
                "SELECT p.*, u.full_name as owner_name 
                 FROM properties p 
                 LEFT JOIN users u ON p.owner_id = u.id 
                 ORDER BY p.created_at DESC"
            );
        } else {
            // 房东：仅获取自己的房产
            $userId = $this->auth->getUserId();
            $stmt = $this->db->prepare(
                "SELECT * FROM properties 
                 WHERE owner_id = ? 
                 ORDER BY created_at DESC"
            );
            $stmt->execute([$userId]);
        }
        
        $properties = $stmt->fetchAll(PDO::FETCH_ASSOC);
        return view('properties/index', ['properties' => $properties]);
    }
    
    public function show($id) {
        if (!$this->auth->isLoggedIn()) {
            redirect('/login');
        }
        
        // 检查权限：管理员或房产所有者
        $stmt = $this->db->prepare("SELECT * FROM properties WHERE id = ?");
        $stmt->execute([$id]);
        $property = $stmt->fetch(PDO::FETCH_ASSOC);
        
        if (!$property) {
            abort(404, '房产不存在');
        }
        
        if (!$this->auth->isAdmin() && $property['owner_id'] != $this->auth->getUserId()) {
            abort(403, '无权访问此房产');
        }
        
        return view('properties/show', ['property' => $property]);
    }
}
```

## 3. 合同管理模块（集成租客信息）

### 3.1 功能概述
管理租赁合同的整个生命周期，合同包含租客信息（非独立用户），适配无电子签名环境。

### 3.2 核心功能
- **合同创建**：基于模板生成合同，包含租客信息
- **合同扫描件**：上传合同扫描件（替代电子签名）
- **合同状态管理**：待签署、已生效、即将到期、已终止
- **自动续约**：到期前自动提醒，一键续约
- **租客信息管理**：作为合同的一部分进行管理
- **权限控制**：管理员查看所有，房东仅查看自己的合同

### 3.3 数据库设计
