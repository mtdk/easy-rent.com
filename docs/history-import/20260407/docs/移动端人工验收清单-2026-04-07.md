# 移动端人工验收清单（M0）

> 文档日期：2026-04-07
> 适用范围：M0 样板页（支付列表、月度对账、财务报表、合同到期提醒、合同列表）
> 目标：在手机常见分辨率下确认页面可读、可操作、无关键回归。

## 一、验收环境

- 设备分辨率：
  - 375x812（iPhone 常见宽度）
  - 390x844（iPhone 新款常见宽度）
  - 412x915（Android 常见宽度）
- 浏览器：Safari（iOS）/ Chrome（Android）
- 网络：正常 4G/5G 与 Wi-Fi 各至少一次

## 二、通用验收项（所有 M0 页面）

- [ ] 页面首屏无横向滚动条
- [ ] 顶部操作区按钮无重叠、无遮挡，可换行展示
- [ ] 表格在小屏下转为卡片化（thead 隐藏，字段标签可见）
- [ ] 每条记录字段标签（data-label）与字段值对应正确
- [ ] 操作按钮可点击，点击区域不小于手指触控常规范围
- [ ] 页面无明显样式错位（边距、圆角、阴影、分割线正常）
- [ ] 页面加载与筛选操作后不出现布局闪烁/重排异常

## 三、页面级验收项

### 3.1 支付列表

- [ ] 两行筛选区在小屏下布局稳定，不挤压输入框
- [ ] 列表卡片中金额、状态、到期日等关键信息清晰可读
- [ ] “查看”按钮可正常打开收据详情
- [ ] 分页区可操作，Previous/Next 与页码跳转正常

### 3.2 月度对账

- [ ] 顶部操作区（筛选、导出、打印等）换行后仍完整显示
- [ ] 对账记录卡片字段顺序符合阅读习惯（账期、金额、状态等）
- [ ] 排序/筛选后结果列表展示正常，无错位

### 3.3 财务报表

- [ ] 各区块报表在小屏下均可读（无关键信息被裁切）
- [ ] 多表格区域均已卡片化，字段标签完整
- [ ] 导出/筛选入口可触达且不误触

### 3.4 合同到期提醒

- [ ] 提醒列表卡片化展示正确
- [ ] 操作按钮在小屏下纵向排布，不重叠
- [ ] 到期日期、租客、房产等核心字段可快速识别

### 3.5 合同列表

- [ ] 顶部按钮区在小屏下可换行，按钮不溢出
- [ ] 合同列表卡片字段标签显示正确（合同编号、租客、房产等）
- [ ] 每条记录操作按钮可点击，跳转页面正确

## 四、回归关注点

- [ ] 不影响桌面端表格展示（>= 992px）
- [ ] 不影响原有筛选、排序、分页、导出流程
- [ ] 不影响权限控制与入口可见性
- [ ] 不引入 PHP 报错或模板渲染异常
