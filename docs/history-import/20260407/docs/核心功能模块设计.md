## 1. 用户认证与权限管理模块

### 1.1 功能概述
提供安全的用户认证、授权和会话管理功能，支持多角色权限控制。

### 1.2 核心功能
- **用户注册**：邮箱/手机号注册，验证码验证
- **用户登录**：账号密码登录，第三方登录（微信、支付宝）
- **权限管理**：基于角色的访问控制(RBAC)
- **会话管理**：JWT token管理，多设备登录控制
- **安全功能**：密码强度检查，登录失败锁定，二次验证

### 1.3 数据库设计
```sql
-- 用户表
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(255) UNIQUE NOT NULL,
    phone VARCHAR(20) UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    full_name VARCHAR(100),
    avatar_url TEXT,
    role VARCHAR(50) DEFAULT 'tenant', -- 'admin', 'landlord', 'tenant'
    is_active BOOLEAN DEFAULT true,
    last_login_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 用户权限表
CREATE TABLE user_permissions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    permission_code VARCHAR(100) NOT NULL,
    granted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 登录日志表
CREATE TABLE login_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    ip_address INET,
    user_agent TEXT,
    login_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    success BOOLEAN DEFAULT true
);
```

### 1.4 API接口设计
```

POST /api/v1/auth/logout        # 用户登出
```

## 2. 房产管理模块

### 2.1 功能概述
管理房东的房产信息，包括房产基本信息、图片、设施、状态等。

### 2.2 核心功能
- **房产信息管理**：CRUD操作，批量导入导出
- **图片管理**：多图上传，图片分类，封面设置
- **地理位置**：地图集成，地址解析，周边设施
- **状态管理**：空闲、已租、维修中、已售等状态
- **设施管理**：房产设施标签管理

### 2.3 数据库设计
```sql
-- 房产表
CREATE TABLE properties (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    owner_id UUID REFERENCES users(id) NOT NULL,
    title VARCHAR(200) NOT NULL,
    description TEXT,
    address TEXT NOT NULL,
    city VARCHAR(100),
    district VARCHAR(100),
    latitude DECIMAL(10, 8),
    longitude DECIMAL(11, 8),
    property_type VARCHAR(50), -- 'apartment', 'house', 'office', 'shop'
    area DECIMAL(10, 2), -- 面积(平方米)
    bedrooms INTEGER,
    bathrooms INTEGER,
    floor INTEGER,
    total_floors INTEGER,
    orientation VARCHAR(50), -- 朝向
    decoration VARCHAR(50), -- 装修情况
    facilities JSONB, -- 设施列表
    status VARCHAR(50) DEFAULT 'vacant', -- 'vacant', 'rented', 'maintenance'
    monthly_rent DECIMAL(12, 2),
    deposit DECIMAL(12, 2), -- 押金
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 房产图片表
CREATE TABLE property_images (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    property_id UUID REFERENCES properties(id) ON DELETE CASCADE,
    image_url TEXT NOT NULL,
    is_primary BOOLEAN DEFAULT false,
    sort_order INTEGER DEFAULT 0,
    uploaded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

-- 房产设施表
CREATE TABLE property_facilities (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    property_id UUID REFERENCES properties(id) ON DELETE CASCADE,
    facility_type VARCHAR(100) NOT NULL, -- 'wifi', 'air_conditioner', 'washing_machine'
```

## 3. 合同管理模块

### 3.1 功能概述
管理租赁合同的整个生命周期，包括创建、签署、执行、续约和终止。

### 3.2 核心功能
- **合同创建**：基于模板生成合同，自定义条款
- **电子签署**：集成第三方电子签名服务
- **合同状态管理**：待签署、已生效、即将到期、已终止
- **自动续约**：到期前自动提醒，一键续约
- **合同模板**：可配置的合同模板系统
- **附件管理**：合同相关文件管理

### 3.3 数据库设计
```sql
-- 合同表
CREATE TABLE contracts (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    property_id UUID REFERENCES properties(id) NOT NULL,
    landlord_id UUID REFERENCES users(id) NOT NULL,
    tenant_id UUID REFERENCES users(id) NOT NULL,
    contract_number VARCHAR(100) UNIQUE NOT NULL,
    title VARCHAR(200) NOT NULL,
    content TEXT NOT NULL, -- 合同内容
    template_id UUID REFERENCES contract_templates(id),
    start_date DATE NOT NULL,
    end_date DATE NOT NULL,
    monthly_rent DECIMAL(12, 2) NOT NULL,
    deposit DECIMAL(12, 2) NOT NULL,
    payment_day INTEGER DEFAULT 1, -- 每月付款日
    status VARCHAR(50) DEFAULT 'draft', -- 'draft', 'pending', 'active', 'expired', 'terminated'
    signed_at TIMESTAMP,
    terminated_at TIMESTAMP,
    termination_reason TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 合同模板表
CREATE TABLE contract_templates (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(200) NOT NULL,
    description TEXT,
    content TEXT NOT NULL, -- 模板内容
    variables JSONB, -- 模板变量定义
    is_default BOOLEAN DEFAULT false,
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 合同签署记录表
CREATE TABLE contract_signatures (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    contract_id UUID REFERENCES contracts(id) ON DELETE CASCADE,
    user_id UUID REFERENCES users(id),
    signature_type VARCHAR(50), -- 'landlord', 'tenant', 'witness'
    signature_data TEXT, -- 签名数据或文件路径
    signed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    ip_address INET,
    user_agent TEXT
);
```

### 3.4 API接口设计
```
GET    /api/v1/contracts                 # 获取合同列表
POST   /api/v1/contracts                 # 创建合同
GET    /api/v1/contracts/{id}            # 获取合同详情
```

## 4. 租金支付模块

### 4.1 功能概述
管理租金的收取、支付记录、账单生成和逾期处理。

### 4.2 核心功能
- **账单生成**：按月自动生成租金账单
- **支付方式**：支持微信支付、支付宝、银行转账
- **支付记录**：完整的支付流水记录
- **逾期管理**：逾期提醒，滞纳金计算
- **收据管理**：电子收据生成和发送
- **对账功能**：自动对账，异常检测

### 4.3 数据库设计
```sql
-- 账单表
CREATE TABLE invoices (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    contract_id UUID REFERENCES contracts(id) NOT NULL,
    invoice_number VARCHAR(100) UNIQUE NOT NULL,
    period_start DATE NOT NULL, -- 账单周期开始
    period_end DATE NOT NULL,   -- 账单周期结束
    amount DECIMAL(12, 2) NOT NULL,
    late_fee DECIMAL(12, 2) DEFAULT 0, -- 滞纳金
    total_amount DECIMAL(12, 2) NOT NULL,
    due_date DATE NOT NULL, -- 到期日
    status VARCHAR(50) DEFAULT 'pending', -- 'pending', 'paid', 'overdue', 'cancelled'
    paid_at TIMESTAMP,
    payment_method VARCHAR(50), -- 'wechat', 'alipay', 'bank_transfer'
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 支付记录表
CREATE TABLE payments (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    invoice_id UUID REFERENCES invoices(id),
    tenant_id UUID REFERENCES users(id),
    amount DECIMAL(12, 2) NOT NULL,
    payment_method VARCHAR(50) NOT NULL,
    transaction_id VARCHAR(200), -- 第三方交易ID
    status VARCHAR(50) DEFAULT 'pending', -- 'pending', 'completed', 'failed', 'refunded'
    paid_at TIMESTAMP,
    confirmed_at TIMESTAMP,
    refunded_at TIMESTAMP,
    metadata JSONB, -- 支付元数据
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 支付配置表
CREATE TABLE payment_configs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    payment_method VARCHAR(50) UNIQUE NOT NULL,
    is_enabled BOOLEAN DEFAULT true,
    config JSONB NOT NULL, -- 支付配置
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4.4 API接口设计
```
GET    /api/v1/invoices                  # 获取账单列表
GET    /api/v1/invoices/{id}             # 获取账单详情
POST   /api/v1/invoices/{id}/pay         # 支付账单
GET    /api/v1/payments                  # 获取支付记录
POST   /api/v1/payments/{id}/refund      # 退款
GET    /api/v1/payment-methods           # 获取支付方式
GET    /api/v1/invoices/overdue          # 获取逾期账单
POST   /api/v1/invoices/batch-generate   # 批量生成账单
```

## 5. 通知提醒模块

### 5.1 功能概述
提供多渠道的通知和提醒功能，包括租金到期提醒、合同到期提醒等。

### 5.2 核心功能
- **多渠道通知**：站内信、短信、邮件、APP推送
- **定时提醒**：基于cron的定时任务调度
- **通知模板**：可配置的通知模板
- **用户偏好**：用户通知偏好设置
- **通知历史**：完整的通知发送记录

### 5.3 数据库设计
```sql
-- 通知表
CREATE TABLE notifications (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) NOT NULL,
    type VARCHAR(50) NOT NULL, -- 'rent_due', 'contract_expiry', 'maintenance', 'system'
    title VARCHAR(200) NOT NULL,
    content TEXT NOT NULL,
    channel VARCHAR(50) DEFAULT 'in_app', -- 'in_app', 'email', 'sms', 'push'
    status VARCHAR(50) DEFAULT 'pending', -- 'pending', 'sent', 'failed', 'read'
    scheduled_at TIMESTAMP, -- 计划发送时间
    sent_at TIMESTAMP,
    read_at TIMESTAMP,
    metadata JSONB,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 通知模板表
CREATE TABLE notification_templates (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(100) NOT NULL,
    type VARCHAR(50) NOT NULL,
    channel VARCHAR(50) NOT NULL,
    subject VARCHAR(200), -- 邮件主题
    content TEXT NOT NULL,
    variables JSONB, -- 模板变量
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 用户通知偏好表
CREATE TABLE user_notification_preferences (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    notification_type VARCHAR(50) NOT NULL,
    channel VARCHAR(50) NOT NULL,
    is_enabled BOOLEAN DEFAULT true,
    UNIQUE(user_id, notification_type, channel)
);
```

### 5.4 API接口设计
```
GET    /api/v1/notifications             # 获取用户通知
POST   /api/v1/notifications             # 发送通知
PUT    /api/v1/notifications/{id}/read   # 标记为已读
DELETE /api/v1/notifications/{id}        # 删除通知
GET    /api/v1/notification-preferences  # 获取通知偏好
PUT    /api/v1/notification-preferences  # 更新通知偏好
POST   /api/v1/notifications/bulk        # 批量发送通知
```

## 6. 报表分析模块

### 6.1 功能概述
提供多维度数据统计和分析功能，帮助房东和管理者做出数据驱动的决策。

### 6.2 核心功能
- **收入统计**：按月、季度、年度收入统计
- **出租率分析**：房产出租率趋势分析
- **租客分析**：租客留存率、满意度分析
- **财务报告**：利润表、现金流量表
- **数据导出**：Excel、PDF格式导出
- **可视化图表**：交互式数据可视化

### 6.3 数据库设计
```sql
-- 统计快照表（用于历史数据分析）
CREATE TABLE statistics_snapshots (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    snapshot_date DATE NOT NULL,
    metric_type VARCHAR(100) NOT NULL, -- 'monthly_income', 'occupancy_rate', etc.
    metric_value DECIMAL(20, 4) NOT NULL,
    dimension JSONB, -- 维度信息
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(snapshot_date, metric_type)
);

-- 报表定义表
CREATE TABLE report_definitions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(200) NOT NULL,
    description TEXT,
    report_type VARCHAR(50) NOT NULL, -- 'financial', 'operational', 'analytical'
    query_sql TEXT, -- 报表查询SQL
    parameters JSONB, -- 报表参数定义
    schedule_cron VARCHAR(100), -- 定时生成cron表达式
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 报表生成记录表
CREATE TABLE report_generations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    report_id UUID REFERENCES report_definitions(id),
    generated_by UUID REFERENCES users(id),
    parameters JSONB, -- 生成时使用的参数
    file_url TEXT, -- 报表文件存储路径
    format VARCHAR(50) DEFAULT 'excel', -- 'excel', 'pdf', 'csv'
    status VARCHAR(50) DEFAULT 'processing', -- 'processing', 'completed', 'failed'
    generated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    completed_at TIMESTAMP
);
```
