# 资源本地化指南

> 文档状态：历史（归档参考）
> 口径说明：以 docs/主任务看板.md 与 docs/任务清单.md 为准。


## 概述

由于收租管理系统需要在无互联网连接的局域网环境中运行，所有外部资源必须下载到本地。本指南详细说明如何获取和配置所需的本地资源。

## 必需资源列表

### 1. Bootstrap 5.3.0
- **CSS文件**: `bootstrap.min.css`
- **JavaScript文件**: `bootstrap.bundle.min.js`
- **映射文件**: `bootstrap.min.css.map`, `bootstrap.bundle.min.js.map` (可选)

### 2. Bootstrap Icons 1.11.0
- **CSS文件**: `bootstrap-icons.css`
- **字体文件**: `bootstrap-icons.woff`, `bootstrap-icons.woff2`
- **映射文件**: `bootstrap-icons.css.map` (可选)

### 3. 可选资源
- **jQuery 3.6.0**: `jquery.min.js` (可选，如需兼容旧代码)
- **Chart.js 4.0+**: `chart.min.js` (可选，用于数据可视化)
- **自定义资源**: 项目特定的CSS和JS文件

## 下载方法

### 方法一：官方CDN下载（需有临时网络）

#### Bootstrap 5.3.0 下载
```bash
# 创建资源目录
mkdir -p public/assets/css public/assets/js

# 下载Bootstrap CSS
wget https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css -O public/assets/css/bootstrap.min.css
wget https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css.map -O public/assets/css/bootstrap.min.css.map

# 下载Bootstrap JS
wget https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js -O public/assets/js/bootstrap.bundle.min.js
wget https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js.map -O public/assets/js/bootstrap.bundle.min.js.map
```

#### Bootstrap Icons 1.11.0 下载
```bash
# 创建字体目录
mkdir -p public/assets/fonts

# 下载Bootstrap Icons CSS
wget https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css -O public/assets/css/bootstrap-icons.css
wget https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css.map -O public/assets/css/bootstrap-icons.css.map

# 下载字体文件
wget https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/fonts/bootstrap-icons.woff -O public/assets/fonts/bootstrap-icons.woff
wget https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/fonts/bootstrap-icons.woff2 -O public/assets/fonts/bootstrap-icons.woff2
```

### 方法二：手动下载（无网络环境）

如果开发环境完全无网络，可以通过以下步骤：

1. **在有网络的机器上**访问以下URL下载文件：
   - Bootstrap: https://github.com/twbs/bootstrap/releases/tag/v5.3.0
   - Bootstrap Icons: https://github.com/twbs/icons/releases/tag/v1.11.0

2. **下载完整包**并解压：
   - Bootstrap: 下载 `bootstrap-5.3.0-dist.zip`
   - Bootstrap Icons: 下载 `bootstrap-icons-1.11.0.zip`

3. **提取所需文件**到对应目录：
   ```
   从bootstrap包中提取：
   - css/bootstrap.min.css → public/assets/css/bootstrap.min.css
   - js/bootstrap.bundle.min.js → public/assets/js/bootstrap.bundle.min.js
   
   从bootstrap-icons包中提取：
   - font/bootstrap-icons.css → public/assets/css/bootstrap-icons.css
   - font/fonts/bootstrap-icons.woff → public/assets/fonts/bootstrap-icons.woff
   - font/fonts/bootstrap-icons.woff2 → public/assets/fonts/bootstrap-icons.woff2
   ```

### 方法三：使用预打包资源

项目可以提供一个 `resources.zip` 文件，包含所有必需的资源，部署时直接解压到 `public/assets/` 目录。

## 目录结构验证

下载完成后，确保目录结构如下：

```
public/assets/
├── css/
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map (可选)
│   ├── bootstrap-icons.css
│   ├── bootstrap-icons.css.map (可选)
│   └── custom.css (项目自定义)
├── js/
│   ├── bootstrap.bundle.min.js
│   ├── bootstrap.bundle.min.js.map (可选)
│   ├── jquery.min.js (可选)
│   ├── chart.min.js (可选)
│   └── app.js (项目自定义)
└── fonts/
    ├── bootstrap-icons.woff
    └── bootstrap-icons.woff2
```

## HTML引用示例

在布局文件中正确引用本地资源：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>收租管理系统</title>
    
    <!-- 本地化Bootstrap CSS -->
    <link href="/assets/css/bootstrap.min.css" rel="stylesheet">
    
    <!-- 本地化Bootstrap Icons -->
    <link href="/assets/css/bootstrap-icons.css" rel="stylesheet">
    
    <!-- 自定义CSS -->
    <link href="/assets/css/custom.css" rel="stylesheet">
    
    <!-- 页面特定CSS -->
    {% block styles %}{% endblock %}
</head>
<body>
    <!-- 页面内容 -->
    
    <!-- 本地化jQuery (可选) -->
    <script src="/assets/js/jquery.min.js"></script>
    
    <!-- 本地化Bootstrap JS -->
    <script src="/assets/js/bootstrap.bundle.min.js"></script>
    
    <!-- 本地化Chart.js (可选) -->
    <script src="/assets/js/chart.min.js"></script>
    
    <!-- 自定义JS -->
    <script src="/assets/js/app.js"></script>
    
    <!-- 页面特定JS -->
    {% block scripts %}{% endblock %}
</body>
</html>
```

## 资源完整性检查

部署前进行资源完整性检查：

1. **文件存在性检查**：
   ```bash
   # 检查必需文件是否存在
   ls -la public/assets/css/bootstrap.min.css
   ls -la public/assets/css/bootstrap-icons.css
   ls -la public/assets/js/bootstrap.bundle.min.js
   ls -la public/assets/fonts/bootstrap-icons.woff
   ```

2. **文件大小验证**：
   - `bootstrap.min.css`: 约 160KB
   - `bootstrap.bundle.min.js`: 约 80KB
   - `bootstrap-icons.css`: 约 100KB
   - `bootstrap-icons.woff`: 约 150KB

3. **浏览器测试**：
   - 在无网络环境下打开页面
   - 检查控制台是否有404错误
   - 验证图标和样式是否正确加载

## 故障排除

### 问题1：图标不显示
- **原因**: 字体文件路径错误
- **解决**: 检查 `bootstrap-icons.css` 中的字体路径，确保指向正确的 `fonts/` 目录

### 问题2：样式错乱
- **原因**: Bootstrap CSS未加载
- **解决**: 检查CSS文件路径和权限

### 问题3：JavaScript功能失效
- **原因**: JS文件未加载或加载顺序错误
- **解决**: 确保Bootstrap JS在jQuery之后加载（如果使用jQuery）

### 问题4：资源加载缓慢
- **原因**: 文件过大或服务器配置问题
- **解决**: 启用Gzip压缩，配置浏览器缓存

## 最佳实践

1. **版本控制**: 将资源文件纳入版本控制
2. **完整性校验**: 部署前验证资源完整性
3. **备份**: 保留资源文件的备份副本
4. **文档**: 记录资源版本和来源
5. **测试**: 在目标环境中测试资源加载

## 自动化脚本

可以创建自动化脚本简化资源下载：

```bash
#!/bin/bash
# download-resources.sh

echo "开始下载本地资源..."

# 创建目录
mkdir -p public/assets/{css,js,fonts}

# 下载Bootstrap
echo "下载Bootstrap 5.3.0..."
wget -q https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css -O public/assets/css/bootstrap.min.css
wget -q https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js -O public/assets/js/bootstrap.bundle.min.js

# 下载Bootstrap Icons
echo "下载Bootstrap Icons 1.11.0..."
wget -q https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css -O public/assets/css/bootstrap-icons.css
wget -q https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/fonts/bootstrap-icons.woff -O public/assets/fonts/bootstrap-icons.woff
wget -q https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/fonts/bootstrap-icons.woff2 -O public/assets/fonts/bootstrap-icons.woff2

echo "资源下载完成！"
```
